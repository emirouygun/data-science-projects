# Data Science Projects Collection

## Overview

This repository contains a collection of hands-on data science projects focusing on:

- Data analysis
- Feature engineering
- Customer segmentation
- Data preprocessing pipelines

Each project is designed to demonstrate practical skills used in real-world data science workflows.

## Projects

### 1. Customer Persona Segmentation

Create customer personas based on demographic and behavioral features and segment them based on revenue.

### 2. RFM Customer Segmentation

Segment customers using Recency, Frequency, and Monetary metrics to analyze purchasing behavior.

### 3. Titanic Feature Engineering

Build a complete data preprocessing pipeline including feature engineering and data cleaning.

## Technologies Used

- Python
- Pandas

- Scikit-learn
- Jupyter Notebook
- Matplotlib
