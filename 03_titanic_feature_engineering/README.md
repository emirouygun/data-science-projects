# Titanic Feature Engineering & Data Pipeline

## Project Overview

This project focuses on building a data preprocessing pipeline using the Titanic dataset.

It includes:

- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data preprocessing functions
- Modular pipeline design

## Objectives

- Understand the dataset
- Handle missing values
- Create new features
- Encode categorical variables
- Prepare data for machine learning models

## Technologies Used

- Python 3
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Project Components

### eda.py

Contains functions for:

- Data overview

### data_prep.py

Includes reusable functions for:

- Missing value handling
- Outlier detection and treatment


## Insights

- Family size may impact survival
- Being alone affects behavior
- Proper preprocessing improves model performance

## Why This Project Matters

- Demonstrates real data pipeline design
- Encourages modular coding
- Shows feature engineering skills
- Prepares data for ML models
