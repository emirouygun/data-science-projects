# RFM Customer Segmentation

## Project Overview

This project applies RFM (Recency, Frequency, Monetary) analysis to segment customers based on their purchasing behavior.

RFM helps answer:

- How recently did a customer purchase? (Recency)
- How often do they purchase? (Frequency)
- How much do they spend? (Monetary)

## Objectives

- Analyze customer purchasing behavior
- Calculate RFM metrics
- Generate RFM scores
- Segment customers into meaningful groups

## Key Concepts

### Recency
Number of days since the customer's last purchase

- Lower value → more active customer

### Frequency

Number of purchases made by the customer

- Higher value → more loyal customer

### Monetary

Total amount spent by the customer

- Higher value → more valuable customer
